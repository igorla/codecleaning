1. Before running the script please replace path to Working Directory
2. The resulted file represent average of each variable for each activity and each subject. See codebook.txt for the details.

Script description:
1) Reads activity_labels.txt and featuresd.txt as dictionaries
2) Read raw features data from train and test folders
3) Combines test/train data
4) Filters mean/std columns
5) Combine activities to raw data
6) Combine Subjects to raw data
7) Set descriptive labels
8) Grouping and summarize the data to get mean for subject and activity
